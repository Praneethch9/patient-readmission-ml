# Patient Readmission Prediction

This project uses a machine learning model to predict whether a patient will be readmitted to the hospital based on clinical and behavioral factors.

## Project Structure
patient_readmission/
│── notebooks/
│   └── patient_readmission.ipynb
│── data/
│   └── patient_data.csv
│── scripts/
│   └── export_to_sqlite.py
│── dashboards/
│── README.md
│── requirements.txt

## Workflow
1. Data preprocessing (Python, Pandas)
2. Train-test split and model building (Logistic Regression)
3. Model evaluation (accuracy, confusion matrix)
4. Store results in SQLite
5. Visualize results (Matplotlib, Seaborn)

## Tech Stack
- Python
- Pandas / Scikit-learn
- SQLite
- Matplotlib / Seaborn
